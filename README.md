# siteportal-i-notes

本仓库用于归档和发布多个独立 HTML 页面，主要面向通用信息展示与笔记整理用途。

## 项目简介

siteportal-i-notes 是一个以 HTML 页面形式存放静态内容的仓库。  
页面内容多为个人笔记、资料整理或功能演示，不绑定特定网站或服务。  
所有页面均为独立文件，可直接通过浏览器打开查看。

## 目录说明

```
siteportal-i-notes/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件为一个独立页面，文件命名应简洁明了。
- `assets/` 目录用于存放页面所需的公共资源，可按需组织子目录。

## 页面归档说明

- 页面内容均以 HTML 格式保存，不依赖后端服务。
- 归档内容不包含任何形式的推广、营销或诱导点击信息。
- 页面之间无强制关联，可单独使用或引用。

## 维护说明

- 本仓库由个人维护，不定期更新内容。
- 欢迎通过 Issues 或 Pull Requests 提交建议或修正。
- 提交新页面时，请确保文件位于 `pages/` 目录下，并保持资源路径清晰。
